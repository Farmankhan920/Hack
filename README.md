# HexaSuite – 75 Developer & Utility Tools

A mobile-first PWA web application with 75 tools, glassmorphism UI, and offline support.

## Quick Start
Open `index.html` in any modern browser, or serve with any static file server:
```bash
npx serve .        # Node.js
python3 -m http.server 8080
```

## Tools (75 total)
- **Encoding/Decoding** (15): Text↔Hex, Base64, URL, HTML, Binary, ASCII, Morse, ROT13
- **Cryptography** (10): SHA-1/256/512, MD5, HMAC, Password Gen/Strength, Token, UUID, Caesar
- **Developer Utilities** (15): JSON/HTML/CSS/JS Formatter, Regex, Colors, Timestamps, Lorem Ipsum, Word Counter, Case Converter, Diff
- **Network** (10): IP Extractor, Header Viewer, JWT Encode/Decode, Subnet Calc, URL/Query Parser, User-Agent, IP/MAC Gen
- **Text Tools** (10): Dedup, Sort, Reverse, Remove Spaces/Specials, Slug, Cleaner, TTS, STT, Char Counter
- **Math & Misc** (15): BMI, Age, EMI, Unit Converter, %, QR, Barcode, Random Number, Stopwatch, Timer, Notes, Todo, Clipboard, File Size, Image Base64

## Tech Stack
- Pure HTML5 / CSS3 / Vanilla JS (no frameworks)
- PWA with Service Worker
- Glassmorphism UI (Inter + Poppins fonts)
- Zero backend required
